Tema 1

¿Qué es el JDK?

Java Development Kit — el conjunto oficial de herramientas para desarrollar, compilar, depurar y ejecutar aplicaciones Java.

Definición

El JDK (Java Development Kit) es el kit de desarrollo oficial de Java. Incluye todo lo necesario para crear aplicaciones: compilador, bibliotecas, herramientas de depuración y el entorno de ejecución.

⚙️

Compilador (javac)

Convierte código Java en bytecode ejecutable.

▶️

Java Runtime (JRE)

Ejecuta aplicaciones Java en cualquier plataforma.

🔧

Herramientas

Depuración, javadoc, jar, profiling y más.

📚

Librerías

Clases y APIs esenciales de Java.

JDK vs JRE — La analogía del conductor

JRE

Solo quieres conducir el auto

  • Ejecutar aplicaciones Java
  • No necesitas crear software
  • Para usuarios finales
vs

JDK

Quieres fabricar automóviles

  • Crear y compilar aplicaciones
  • Incluye el JRE + herramientas de desarrollo
  • Para desarrolladores
Java SE

Standard Edition

🚗 El automóvil familiar

Edición estándar para aplicaciones de escritorio, herramientas y programas generales. Punto de partida de todo desarrollador Java.

Java EE

Enterprise Edition

🚛 El camión empresarial

Para bancos, aerolíneas, hospitales y sistemas que atienden millones de usuarios. Escalable, seguro y de alto rendimiento.

Java ME

Micro Edition

🏍️ La motocicleta ligera

Para dispositivos con pocos recursos: IoT, sistemas embebidos y equipos pequeños.

Tema 2

Tipos de Datos en Java

Java es un lenguaje fuertemente tipado: cada dato debe declararse con un tipo específico, como cajas de almacenamiento de distintos tamaños.

Ejemplo
int edad = 25;
double precio = 19.99;
char inicial = 'A';
boolean activo = true;
String nombre = "Java";

Tema 3

Visual Studio Code vs Cursor AI

Dos cocinas: una tradicional donde cocinas manualmente, y otra inteligente con un asistente que te ayuda en cada paso.

📝

Visual Studio Code

La navaja suiza para desarrolladores

🍳 Cocina tradicional — todos los utensilios, pero haces casi todo manualmente.

  • Editor ligero, rápido y extensible
  • Autocompletado inteligente
  • Depuración avanzada e integración Git
  • Miles de extensiones disponibles
  • 100% personalizable y gratuito
Ideal para: Control total, equipos con muchas tecnologías, proyectos que requieren extensiones específicas.
✨

Cursor AI

El copiloto inteligente

🤖 Cocina inteligente — un asistente te recomienda, corrige errores y acelera tu trabajo.

  • Basado en VS Code + IA integrada
  • Chat con tu código (AI Chat)
  • Generación y refactorización con IA
  • Edición multi-archivo contextual
  • Detección y corrección de errores
Ideal para: Máxima productividad, automatización, proyectos complejos y generación rápida de código.

¿Qué es un AI Engineer?

Un profesional que diseña, desarrolla e implementa soluciones basadas en Inteligencia Artificial para resolver problemas reales.

"Un AI Engineer es un profesional que diseña, desarrolla e implementa soluciones basadas en Inteligencia Artificial para resolver problemas reales y generar valor."
💡

Diseña soluciones

Crea soluciones de IA que resuelven necesidades del negocio.

🧩

Integra modelos

Conecta modelos de IA en productos, apps y procesos.

📊

Trabaja con datos

Entrena, evalúa y mejora modelos con datos reales.

⚙️

Despliega sistemas

Construye, prueba y despliega sistemas inteligentes escalables.

¿Cómo trabaja un AI Engineer?

1
Comprende el problema

Analiza la necesidad y define objetivos.

2
Prepara los datos

Recopila, limpia y organiza información.

3
Diseña la solución

Selecciona modelos y define estrategia.

4
Desarrolla

Construye, entrena e integra la solución.

5
Despliega

Lleva la solución a producción.

6
Monitorea y mejora

Evalúa resultados y optimiza continuamente.

¿Dónde se aplica?

Automatización de procesos Asistentes inteligentes Análisis predictivo Recomendaciones personalizadas Detección de fraudes Generación de contenido
